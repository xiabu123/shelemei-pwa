# SLM / 射了没 版本总目录

这是 `射了没` 项目的公开归档根目录，后续会以这个仓库作为持续迭代的起点。

这是当前 `v0.01bata` 的归档与迭代根目录。

## 目录说明

- `docs/`
  - 产品文档、PRD、设计说明
  - 以及本次版本的迭代复盘
  - 适合后续补充需求、结构变化、版本说明
- `source/`
  - 当前版本的源码快照
  - `source/shelemei-pwa/` 是这一版的主工程
- `release/`
  - 当前版本发布件
  - 包含 `v0.01bata.pkg`、校验文件、安装说明、发布清单
- `builds/`
  - 当前版本的构建产物
  - `web/` 是 PWA 构建结果
  - `android/` 是 Android 安装包与构建日志
- `references/`
  - 早期设计稿、PRD、Android 说明等历史参考资料

## 当前版本

- 版本号：`v0.01bata`
- 方向：手机优先 PWA + Android 封装
- 当前重点：快捷记录、首页数据概览、AI 射分析、历史记录、本地持久化
- 开源许可：MIT

## 你后续可以这样继续

- 新版本需求：放到 `docs/` 里按日期追加
- 新源码迭代：继续放在 `source/` 下的新版本分支或子目录
- 新构建包：放在 `release/` 和 `builds/` 对应版本目录中

## 本次迁移说明

我已经把这版真正相关的内容整理进来，源码、文档、发布包和构建产物都分开了。像原工作区里一些不相关的临时文件，我没有一并搬过来，避免把后面迭代环境弄乱。

## 公开仓库的使用感

这个仓库会把“能跑起来的版本”和“怎么迭代它的资料”一起保留下来。对外展示时，它适合作为一个完整的开源起点；对内迭代时，它也适合继续往下长出新版本。
